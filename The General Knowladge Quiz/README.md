# The General Knowledge Quiz

Welcome to **The General Knowledge Quiz**! This is a simple, interactive general knowledge quiz project created to demonstrate control flow (if-else logic), variables, and user input handling. 

The project contains two versions:
1. **Interactive Web Application** (HTML/CSS/JavaScript)
2. **Terminal Console Game** (Python CLI)

---

## 🎮 How to Play

### 1. Web Version (Interactive UI)
This version features a modern glassmorphism design, responsive layouts, satisfying animations, and dynamic progress/score tracking.

#### Option A: Open Directly
1. Navigate to the project directory: `d:\The General Knowladge Quiz`
2. Double-click the `index.html` file to open it directly in your web browser (Chrome, Edge, Firefox, etc.).

#### Option B: Open with VS Code "Live Server"
1. Open VS Code and open this project folder.
2. Install the **Live Server** extension (by Ritwick Dey) from the Extensions tab (`Ctrl` + `Shift` + `X`).
3. Open `index.html` in VS Code and click **Go Live** at the bottom-right of the status bar.

---

### 2. Python Version (Terminal Console)
This version is a classic terminal-based program designed with Unicode-safe formatting for Windows compatibility.

#### How to Run:
1. Open your terminal (PowerShell, Command Prompt, or VS Code Terminal).
2. Run the following command:
   ```bash
   python "quiz.py"
   ```

---

## 🛠️ Project Structure

* **`index.html`**: The structure of the web application.
* **`style.css`**: Premium styling, colors, and layout configurations.
* **`app.js`**: Frontend game logic, score evaluations, and page transitions.
* **`quiz.py`**: Console-based terminal game implementation.

---

## 🚀 Key Learning Concepts

* **Variables**: Keeping track of state (such as the player's score).
* **If-Else Control Flow**: Evaluating input choices and rendering different outcomes based on user answers.
* **Responsive Design**: Structuring layout containers for both mobile and desktop screens.
