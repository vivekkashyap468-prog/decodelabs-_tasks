# Simple To-Do List

A simple, interactive command-line To-Do List application written in Python. This project demonstrates foundational Python concepts including lists, loops, user inputs, and basic error handling.

## Features
- **Add Tasks**: Quickly append tasks to your list.
- **View Tasks**: Print all current tasks with their numbered indices.
- **Remove Tasks**: Remove specific tasks from the list using their corresponding numbers.
- **Input Validation**: Ensures valid selections and inputs are entered.

## How to Run

1. Make sure you have [Python](https://www.python.org/) installed.
2. Open your terminal (Command Prompt or PowerShell).
3. Run the following command:
   ```bash
   python Todolist.py
   ```

## Key Python Skills Used
- **Lists**: Storing and managing multiple items.
- **While Loops**: Keeping the program running until the user decides to exit.
- **Control Flow**: Using `if-elif-else` conditions for user choice selection.
- **Enumerate**: Printing tasks along with their line numbers.
